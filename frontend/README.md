# Frontend

React + Vite frontend for the social network.

## Quick Start

```bash
# Install dependencies
npm install

# Configure environment
cp .env.example .env

# Start development server
npm run dev
```

The frontend will run on `http://localhost:3000`

## Build

```bash
npm run build
npm run preview
```

## Architecture

- **Framework**: React 18
- **Build tool**: Vite
- **HTTP client**: Axios
- **Styling**: CSS modules

## Features

- User authentication (login/signup)
- Feed with posts
- Post creation with media support
- Post reactions

## Design

- Primary colors: Green (#10b981) and white
- Minimalist layout
- Responsive design
- Clean typography
